# Handwritten-Digit-Recognition
Image classification project using machine learning to recognize handwritten digits from pixel data.

## Problem Statement
Predict handwritten digit labels from image pixel intensity values.

## Workflow
- Data loading and preprocessing
- Visualization of digit samples
- Model training and evaluation
- Performance assessment

## Models Used
- Logistic Regression
- KNN
- Random Forest

## Tools
Python, NumPy, Pandas, Scikit-learn, Matplotlib
